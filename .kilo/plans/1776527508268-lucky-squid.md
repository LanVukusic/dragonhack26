# Plan: Typesafe WebSocket Interface and Circle Rendering

## Goal
Implement a typesafe TypeScript interface for WebSocket events and a separate circle rendering component.

## Current State
- `web/src/App.tsx` has partial WebSocket setup but incomplete
- Uses `react-konva` for canvas rendering
- WebSocket connects to `ws://localhost:8000/ws`

## Implementation Steps

### 1. Create types file
**File**: `web/src/types/websocket.ts`
- Define `CircleData` interface with: `id`, `x`, `y`, `rotation`, `color` (as tuple `[number, number, number]`)
- Define `WebSocketMessage` interface with: `timestamp`, `circles`
- Export both with proper TypeScript typing

### 2. Create circle rendering component
**File**: `web/src/components/CircleRenderer.tsx`
- Accept `circles: CircleData[]` as props
- Render each circle using `react-konva`'s `<Circle>`
- Display position (`x`, `y`), `rotation`, and `color` (convert RGB tuple to CSS color string)
- Support dragging (optional)

### 3. Update App.tsx
- Import types from new types file
- Import `CircleRenderer` component
- Use state to store circles from WebSocket
- Render `CircleRenderer` with received data

## Data Format
```json
{
  "timestamp": "2026-04-18T...",
  "circles": [
    {"id": 0, "x": 150.3, "y": 220.1, "rotation": 45.0, "color": [255, 128, 64]},
    ...
  ]
}
```

## Color Conversion
Convert `[r, g, b]` tuple to CSS: `rgb(r, g, b)`
